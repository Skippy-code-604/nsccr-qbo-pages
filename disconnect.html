Disconnect

You can revoke the NSCCR QuickBooks Connector's access to QuickBooks Online at any time. Because the App is self-hosted, disconnecting is done in two places:

1. Revoke access in QuickBooks / Intuit

  1. Sign in to QuickBooks Online.
  2. Go to Settings (gear) → Apps (or your Intuit account's Connected apps / authorized apps).
  3. Find this app and choose Disconnect. This invalidates the App's tokens.

2. Remove local credentials

  1. Stop the server if it is running.
  2. Delete the local tokens.json file (the stored access/refresh tokens).
  3. Remove the credential values from .env if you want to fully clear them.

After these steps the App can no longer access any QuickBooks data until it is reconfigured and re-authorized. This page exists to satisfy the Disconnect URL field in the Intuit Developer app configuration.
