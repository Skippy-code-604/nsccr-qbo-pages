Launch

The NSCCR QuickBooks Connector is a private, self-hosted application — there is no hosted web app to sign in to. It runs locally on the operator's own computer and is launched from a command line.

How it runs

  1. Configure credentials in the local .env file.
  2. Start the server: npm start.
  3. It connects to the authorized QuickBooks Online company through Intuit's OAuth 2.0 flow and operates over the standard QuickBooks Online API.

Because the App is single-company and self-hosted, this page exists to satisfy the Launch URL field in the Intuit Developer app configuration; it is not an interactive sign-in page.

To revoke access, see the Disconnect page.
